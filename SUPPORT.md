# Noden Support

Before opening an issue, check the [Noden documentation](https://noden.useroamteknoloji.com/docs.html).

For bugs, include your Noden version, macOS version, Mac architecture, authentication method and clear reproduction steps. Never include passwords, private keys, API keys or production server credentials.

For billing, licensing or private support, email **noden@useroamteknoloji.com**.
